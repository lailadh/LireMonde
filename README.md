# LireMonde

## Qu'est-ce que LireMonde ?

LireMonde est une plateforme web simple pour parcourir et gérer des livres. Elle permet aux utilisateurs de :

- Découvrir de nouveaux livres
- Rechercher des livres
- Construire leur propre liste de lecture personnelle

## Pages principales

- **Accueil** — Page d'accueil avec affichage des livres, filtrage par genre et recherche
- **À lire** — Liste de lecture personnelle, contenant les livres ajoutés aux favoris
- **Admin** — Tableau de bord pour ajouter, modifier et supprimer des livres

## Technologies utilisées

- **HTML / CSS / JavaScript** — Interface frontale
- **JSON Server** — API et base de données (REST API)

## Fonctionnalités

Recherche de livres (par titre ou auteur)
Filtrage par genre littéraire
❤️ Ajout de livres à la liste "À lire"
Fenêtre modale avec les détails du livre
➕ Ajouter des livres (Admin)
✏️ Modifier des livres (Admin)
🗑️ Supprimer des livres (Admin)
